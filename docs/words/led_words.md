# LED Words

## STM32L476 LED Words

##### `led-red-on`
( -- )

Turn the red LED on

##### `led-red-off`
( -- )

Turn the red LED off

##### `led-green-on`
( -- )

Turn the green LED on

##### `led-green-off`
( -- )

Turn the green LED off
